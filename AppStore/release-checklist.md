# Mira Release Checklist

- [ ] XcodeGen project generated.
- [ ] XcodeBuildMCP build passes with zero warnings.
- [ ] Focused tests pass.
- [ ] App Store metadata matches shipped features.
- [ ] Privacy/support docs are current.
- [ ] Screenshots are captured from the real app.
- [ ] No secrets or private endpoint data are staged.
- [ ] Version/build updated with `xcode-versioning`.
- [ ] Release notes updated.
- [ ] Commit and push complete.
- [ ] TestFlight upload complete when release-gated.
