## Reference App Lessons

LM Manager Pro is a one-way reference for tested patterns: SwiftUI MVVM,
Keychain-backed provider keys, route selection, generated media storage outside
SwiftData, endpoint clients, task ledgers, release notes, and TestFlight
workflow. Mira should borrow lessons, not inherit the large single-app schema.
